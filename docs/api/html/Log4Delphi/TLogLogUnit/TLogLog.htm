Class
  TLogLog

Class Hierarchy
  TObject

Related Unit
  TLogLogUnit

Description
 

This class is used internally to perfrom logging within the Log4Delphi
package. It is not meant to be used outside the Log4Delphi package.
Typically statements are logged to a file named 'log4delphi.log'.

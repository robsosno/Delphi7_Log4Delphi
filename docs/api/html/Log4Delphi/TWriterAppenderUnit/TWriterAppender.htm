Class
  TWriterAppender

Class Hierarchy
  TAppender

Related Unit
  TWriterAppenderUnit

Description
 

TWriterAppender appends log events to a TWriter. This can be used in
combination with streams thus allowing logging to file streams, network
streams or other stream based resources.

Class
  TLoggingEvent

Class Hierarchy
  TObject

Related Unit
  TLoggingEventUnit

Description
 

The internal representation of logging events. When an affirmative decision
is made to log then a LoggingEvent instance is created. This instance is
passed around to the different log4delphi components.
